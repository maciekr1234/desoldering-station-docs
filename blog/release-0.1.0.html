Release 0.1.0

December 27, 2025

We are excited to announce the release of version 0.1.0!

Featuresâ€‹

  • Enable logging in LVGL configuration, update build version, and enhance ADC task creation error handling
  • Update heater and pump settings, enhance configuration schema, and refactor related components
  • Enhance heater settings and add settings menu functionality
  • Implement temperature display widget
  • Add ADC driver and temperature sensor classes
  • Enhance LVGL integration and add logging functionality
  • Add fan control functionality with configurable temperature curves

Bug Fixesâ€‹

  • Remove redundant note about version bumps in git commit guidelines
  • Add 'pid-calibration' to the sidebar for improved documentation navigation
  • Clarify overshoot description in PID calibration guide
  • Change publish branch from main to master in documentation workflow

  • Fix empty deprecated tag causing Doxygen plugin crash
  • Fix msgbox creation and API compatibility with LVGL 9.4

Documentationâ€‹

  • Release v0.1.0 with complete documentation versioning
  • Add Docusaurus documentation site with versioned releases
  • Add comprehensive architecture documentation with diagrams
  • Add PID calibration guide with tuning methodology
  • Add auto-tuning guide for automatic PID parameter discovery
  • Add contributing guidelines and versioning strategy
  • Add API reference auto-generated from Doxygen comments

Controllers & Driversâ€‹

  • Add HeaterController with PID temperature control and state machine

  • Add motion sensor for iron detection
