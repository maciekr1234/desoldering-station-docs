Motor Driver

H-bridge motor driver for the vacuum pump.

Overviewâ€‹

The MotorDriver class controls a brushed DC motor through an H-bridge (DRV8833 or similar). It supports speed control via PWM with smooth acceleration ramping, bidirectional operation, and fault detection.

Featuresâ€‹

  • PWM speed control (-1.0 to +1.0)
  • Smooth acceleration ramping with configurable duration
  • Direction control (forward/reverse)
  • Brake, coast, enable/disable modes
  • Fault detection and recovery
  • Command timeout protection

Speed changes are smoothly ramped over the configured acceleration time:

Example Usageâ€‹

#include "drivers/motor_driver.h"

// Create with individual pins
MotorDriver motor(14, 15, 13, 12, 20000);

// Or from config
motor_driver_config_t config = {
    .in1 = 14, .in2 = 15,
    .sleep_pin = 13, .fault_pin = 12,
    .freq_hz = 20000
};
MotorDriver motor(config);

void setup() {
    motor.setAccelerationTime(500);    // 500ms ramp
    motor.setCommandTimeout(2000);      // 2s timeout
}

void start_pump() {
    motor.enable();
    motor.speed(1.0f);  // Full speed forward with ramp
}

void start_pump_immediate() {
    motor.enable();
    motor.speed(1.0f, true);  // Full speed immediately
}

void stop_pump() {
    motor.coast();  // Coast to stop
    motor.disable();
}

void reverse_pump() {
    motor.speed(-0.5f);  // Half speed reverse
}

Fault Handlingâ€‹

if (motor.getFault()) {
    PICO_LOGE("MOTOR", "Fault detected!");
    
    // Attempt recovery
    if (motor.recoverFromFault(3)) {
        PICO_LOGI("MOTOR", "Recovered from fault");
    } else {
        PICO_LOGE("MOTOR", "Failed to recover");
    }
}

Command Timeoutâ€‹

If no speed command is received within command_timeout_ms, the motor automatically coasts and disables to prevent runaway operation.
