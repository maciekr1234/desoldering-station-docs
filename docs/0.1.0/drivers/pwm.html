PWM Driver

Hardware PWM abstraction for the RP2040.

Overviewâ€‹

The PWM driver wraps the RP2040's hardware PWM slices, providing a simple interface for setting duty cycles and frequencies. Used by the heater controller and motor driver.

Featuresâ€‹

  • High resolution duty cycle control
  • Configurable frequency
  • 8 PWM slices, 16 channels
  • Duty cycle specified as float (0.0 to 1.0)

API Referenceâ€‹

Typesâ€‹

typedef struct {
    uint8_t  pin;       // GPIO pin number
    uint8_t  slice;     // PWM slice number (0-7)
    uint8_t  channel;   // Channel within slice (A=0, B=1)
    uint32_t frequency; // Frequency in Hz
    float    duty;      // Duty cycle (0.0 to 1.0)
} pwm_handle_t;

Functionsâ€‹

#include "drivers/pwm.h"

// Initialize PWM on a pin
// Returns: PICO_OK on success, error code otherwise
int hw_pwm_init(uint8_t pin, uint32_t frequency_hz, pwm_handle_t* handle);

// Deinitialize PWM
// Returns: PICO_OK on success, error code otherwise
int hw_pwm_deinit(pwm_handle_t* handle);

// Enable/disable PWM output
void hw_pwm_enable(pwm_handle_t* handle, bool enabled);

// Set PWM frequency
// Returns: PICO_OK on success, error code otherwise
int hw_pwm_set_frequency(pwm_handle_t* handle, uint32_t frequency_hz);

// Set duty cycle (0.0 to 1.0)
// Returns: PICO_OK on success, error code otherwise
int hw_pwm_set_duty(pwm_handle_t* handle, float duty);

// Set PWM period in milliseconds
// Returns: PICO_OK on success, error code otherwise
int hw_pwm_set_period(pwm_handle_t* handle, uint32_t period_ms);

Pin to Slice Mappingâ€‹

Each slice has two channels (A and B) that share the same frequency but can have independent duty cycles.

Example Usageâ€‹

#include "drivers/pwm.h"

pwm_handle_t heater_pwm;

void setup() {
    // Initialize 25kHz PWM on GPIO 6
    if (hw_pwm_init(6, 25000, &heater_pwm) == PICO_OK) {
        hw_pwm_set_duty(&heater_pwm, 0.0f);  // Start at 0%
        hw_pwm_enable(&heater_pwm, true);
    }
}

void set_heater_power(float power) {
    // power: 0.0 to 1.0
    hw_pwm_set_duty(&heater_pwm, power);
}
