Temperature Driver

Temperature sensor driver for heater (PTC) and ambient (NTC) temperature measurement.

Overviewâ€‹

The Temperature driver provides functions to read two different temperature sensors:

  • Heater PTC sensor: Uses a lookup table with linear interpolation for accurate readings
  • Ambient NTC sensor: Uses the Beta equation for thermistor calculations

Featuresâ€‹

  • PTC heater sensor with lookup table interpolation
  • NTC ambient sensor with Beta equation
  • Multi-point calibration for heater temperature
  • Handle disconnection detection
  • No initialization required (uses ADC driver)

Example Usageâ€‹

#include "drivers/temperature.h"
#include "core/pico_log.h"

void read_temperatures() {
    // Read raw heater temperature
    float heater_temp = read_heater_temperature_celsius();
    
    if (heater_temp == -1000.0f) {
        PICO_LOGW("TEMP", "Handle disconnected!");
        return;
    }
    
    // Read calibrated heater temperature
    float calibrated_temp = read_heater_temperature_filtered();
    
    // Read ambient temperature
    float ambient_temp = read_ambient_temperature_celsius();
    
    PICO_LOGI("TEMP", "Heater: %.1fÂ°C (cal: %.1fÂ°C), Ambient: %.1fÂ°C",
              heater_temp, calibrated_temp, ambient_temp);
}

Error Handlingâ€‹

float temp = read_heater_temperature_celsius();

if (temp == -1000.0f) {
    // Handle disconnected - high voltage detected
    PICO_LOGE("TEMP", "No handle connected!");
} else if (temp < 25.0f || temp > 500.0f) {
    // Reading clamped to lookup table bounds
    PICO_LOGW("TEMP", "Temperature out of calibrated range");
}
