triggerActivity()â€‹

void HeaterController::triggerActivity ()

Trigger activity to reset standby/sleep timers.

This method should be called when user activity is detected (e.g., pump trigger, encoder rotation, touch). It resets the inactivity timers and wakes the heater from sleep/standby states if appropriate.
