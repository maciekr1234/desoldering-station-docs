Descriptionâ€‹

PID auto-tuning using relay feedback method with thermal lag identification.

Implements the relay feedback auto-tuning method to automatically determine optimal PID parameters and thermal lag compensation for the heater system.

Usage:

  • Call auto_tune_start() with configuration
  • Periodically call auto_tune_update() with current temperature (returns power %)
  • Check auto_tune_get_state() for completion
  • Get results with auto_tune_get_result()
info

The tuning runs in the heater's Tuning state and controls power output.
