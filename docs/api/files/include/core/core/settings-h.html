Application settings structures and definitions.

Defines the data structures for storing heater, pump, and system settings, along with default values and versioning.

enum heater_startup_state_t

Heater state at startup.

Enumeration values
HEATER_STARTUP_SLEEP

Heater stays in sleep mode at startup (= 0)

HEATER_STARTUP_STANDBY

Heater enters standby at startup (= 1)

HEATER_STARTUP_ON

Heater starts heating immediately at default temperature (= 2)
