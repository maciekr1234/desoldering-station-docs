lv_settings_numeric_bind_value()â€‹

lv_observer_t * lv_settings_numeric_bind_value (lv_obj_t * obj, lv_subject_t * subject)

Bind a numeric settings item to a subject.

Supports both LV_SUBJECT_TYPE_INT and LV_SUBJECT_TYPE_FLOAT:

  • INT: Direct mapping of internal fixed-point value
  • FLOAT: Automatic conversion using decimal_places (e.g., internal 1234 with decimal_places=2 maps to float 12.34)

When the numeric value changes, the subject is updated. When the subject changes, the numeric value is updated to match.

Parameters
objPointer to the numeric settings item object.
subjectPointer to a subject (LV_SUBJECT_TYPE_INT or LV_SUBJECT_TYPE_FLOAT).
Returns

Pointer to the created observer, or NULL on failure.
