structlv_temperature_display_t

Temperature display object structure. This object displays the current temperature and allows setting a setpoint temperature. It includes labels for current temperature and setpoint, a subject for observing setpoint changes, step values for incrementing/decrementing the setpoint, and a timer for edit mode. More...

lv_observer_t * lv_temperature_display_bind_temperature (lv_obj_t * obj, lv_subject_t * subject)

Bind a temperature subject to the temperature display. The display will update automatically when the subject's value changes.

Parameters
objPointer to the temperature display object.
subjectPointer to the temperature subject to bind.
Returns

Pointer to the created observer.

info

The subject should be of type LV_SUBJECT_TYPE_INT or LV_SUBJECT_TYPE_FLOAT.
