Heater controller interface.

Provides an interface for controlling a heating element with PID control, state management, error handling, and callback registration.

enum class HeaterError
strong

Heater error codes for fault detection.

Enumeration values
None

No error (= 0)

NoHandle

Handle not connected (open circuit)

SensorFailure

Temperature sensor reading invalid

Overtemperature

Temperature exceeded maximum safe limit

HeatingTimeout

Failed to reach target temperature in time

RunawayHeating

Temperature rising uncontrollably

UnderTemperature

Temperature dropped unexpectedly while heating

enum class HeaterState
strong

Heater operational states.

Enumeration values
Sleep

Heater is off/sleeping

Standby

Heater at standby temperature

Running

Heater actively heating to target

Error

Error state - heater disabled

Tuning

PID tuning mode - manual control

Defines the various states the heater can be in during operation.
