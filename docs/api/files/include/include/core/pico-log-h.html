Enumerationsâ€‹

enumâ€‹

enum log_level_t

Log level enumeration. Defines the severity levels for log messages.

Enumeration values
LOG_LEVEL_NONE

No log output (= 0)

LOG_LEVEL_ERROR

Critical errors, software module can not recover on its own (= 1)

LOG_LEVEL_WARN

Error conditions from which recovery measures have been taken (= 2)

LOG_LEVEL_INFO

Information messages which describe normal flow of events (= 3)

LOG_LEVEL_DEBUG

Extra information which is not necessary for normal use (values, pointers, sizes, etc) (= 4)

LOG_LEVEL_VERBOSE

Bigger chunks of debugging information, or frequent messages which can potentially flood the output (=5)

LOG_LEVEL_MAX

Number of levels supported (= 6)
