Descriptionâ€‹

PID controller driver interface.

Provides functions to create, configure, and use PID controllers for closed-loop control systems.

Enumerationsâ€‹

enumâ€‹

enum pid_calculate_type_t

PID calculation type enumeration.

Enumeration values
PID_CAL_TYPE_INCREMENTAL

PID calculation based on error changes (Î”error)

PID_CAL_TYPE_POSITIONAL

PID calculation based on absolute error

This enumeration defines the two types of PID calculation methods:

  • PID_CAL_TYPE_INCREMENTAL: Computes the PID output based on changes in error (Î”error). This method is suitable for systems where only changes to the output are applied, rather than setting the full output each cycle.
  • PID_CAL_TYPE_POSITIONAL: Computes the PID output based on the absolute error value. This method is suitable for systems where the full output is set each cycle based on the current error.
