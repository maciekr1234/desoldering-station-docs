Descriptionâ€‹

Temperature display widget for LVGL.

This widget displays the current temperature and allows setting a desired setpoint temperature. It supports incrementing and decrementing the setpoint using defined step values. The widget can also bind to a temperature subject to automatically update the displayed temperature.

Bind a temperature subject to the temperature display. The display will update automatically when the subject's value changes.

Parameters

static

Show or hide the setpoint display.

Parameters
objPointer to the temperature display object.
showTrue to show the setpoint, false to show the current temperature.

When showing the setpoint, the current temperature label is hidden and vice versa. The edit timer is reset when showing the setpoint and paused when hiding it.
