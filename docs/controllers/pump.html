Pump Controller

Vacuum pump motor control with trigger debouncing.

Overviewâ€‹

The PumpController manages the vacuum pump motor, handling trigger input debouncing, motor speed control, and integration with the heater system for coordinated operation.

Featuresâ€‹

  • Hardware trigger debouncing
  • Soft start/stop motor control
  • Speed configuration
  • Callback support for state changes
  • IRQ-based trigger detection

API Referenceâ€‹

Class Definitionâ€‹

class PumpController {
public:
    using trigger_callback_t = std::function<void(bool pressed)>;
    
    // Debounce states
    enum class DebounceState {
        IDLE,       // Waiting for trigger
        DEBOUNCING, // Validating press/release
        STABLE      // Confirmed state
    };
    
    static constexpr uint32_t DEBOUNCE_DELAY_MS = 50;
    
    PumpController(pin_size_t trigger_pin, motor_driver_config_t motor_cfg);
    ~PumpController();
    
    // Initialization
    bool begin(pin_size_t trigger_pin);
    bool begin();
    void end();
    
    // Update (call from task)
    void update();
    
    // Trigger state
    bool readTrigger();
    bool isTriggerPressed() const;
    
    // Configuration
    void setTriggerPin(pin_size_t pin);
    pin_size_t getTriggerPin() const;
    void setTriggerCallback(trigger_callback_t cb);
    
    // Motor access
    MotorDriver& getMotorDriver();
};

Trigger Handlingâ€‹

Hardware Setupâ€‹

The trigger is connected to a GPIO with internal pullup:

    VCC
     |
    [R] Internal Pullup
     |
    GPIO ---+--- To MCU
            |
           [ ] Trigger Switch
            |
           GND
  • Idle: GPIO reads HIGH (pullup)
  • Pressed: GPIO reads LOW (grounded)

IRQ Handlerâ€‹

// ISR in RAM for flash safety
__not_in_flash_func(void _pump_controller_irq)(void* vptr) {
    auto* pump = static_cast<PumpController*>(vptr);
    // Set flag for task processing
    pump->_changed = true;
}

Example Usageâ€‹

#include "pump.h"

motor_driver_config_t motor_cfg = {
    .in1_pin = 2,
    .in2_pin = 3,
    .pwm_freq = 20000,
    .accel_time_ms = 500
};

PumpController pump(15, motor_cfg);

void setup() {
    pump.begin();
    
    // Optional: get notified of trigger changes
    pump.setTriggerCallback([](bool pressed) {
        PICO_LOGI("PUMP", "Trigger %s", pressed ? "pressed" : "released");
    });
    
    // Configure motor
    pump.getMotorDriver().setAccelerationTime(300);
}

void pump_task() {
    while (true) {
        pump.update();
        vTaskDelay(pdMS_TO_TICKS(10));
    }
}
