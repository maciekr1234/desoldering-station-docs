ADC Driver

Timer-driven ADC (Analog-to-Digital Converter) driver with automatic periodic sampling and filtering.

Overviewâ€‹

The ADC driver provides hardware abstraction for the RP2040's 12-bit ADC with automatic periodic sampling. It uses a repeating timer to read ADC values at regular intervals, applies filtering (circular buffer averaging with outlier rejection), and stores results in a thread-safe manner using FreeRTOS mutex protection.

Featuresâ€‹

  • Timer-driven periodic sampling (configured via CONFIG_ADC_TIMER_PERIOD_MS)
  • Circular buffer averaging with outlier rejection
  • FreeRTOS mutex protection for thread safety
  • Raw, filtered, and millivolt output for each channel
  • Dynamic channel configuration based on config.h pin definitions

Architectureâ€‹

Channel Configurationâ€‹

Channels are dynamically configured based on pin definitions in config.h:

Channel EnumConfig PinPurpose
ADC_INPUT_AMBIENT_TEMPCONFIG_PIN_AMBIENT_TEMPNTC ambient sensor
ADC_INPUT_PTC_TEMPCONFIG_PIN_HEATER_TEMPPTC heater sensor
ADC_INPUT_VREFCONFIG_PIN_VREFVoltage reference

Channels are only included if their corresponding pin is not set to 255.

API Referenceâ€‹

Typesâ€‹

// Channel identifier type
typedef uint32_t adc_input_channel_t;

// Channel data structure
typedef struct {
    uint16_t adc_raw;      // Raw ADC value (0-4095)
    uint16_t adc_filtered; // Filtered ADC value
    uint16_t millivolts;   // Converted millivolt value
} adc_input_channel_data_t;

Functionsâ€‹

#include "drivers/adc.h"

// Initialize ADC and start periodic sampling timer
void adc_input_init(void);

// Stop sampling and deinitialize ADC
void adc_input_deinit(void);

// Get millivolt value for a channel
uint16_t adc_input_get_millivolts(adc_input_channel_t channel);

// Get ADC value (raw or filtered)
uint16_t adc_input_get_adc(adc_input_channel_t channel, bool filtered);

// Get full channel data structure
int adc_input_get_channel(adc_input_channel_t channel, adc_input_channel_data_t* out_data);

Example Usageâ€‹

#include "drivers/adc.h"
#include "core/pico_log.h"

void setup() {
    // Initialize ADC with periodic sampling
    adc_input_init();
}

void read_sensors() {
    // Get filtered ADC value
    uint16_t ptc_adc = adc_input_get_adc(ADC_INPUT_PTC_TEMP, true);
    
    // Get millivolt reading
    uint16_t ptc_mv = adc_input_get_millivolts(ADC_INPUT_PTC_TEMP);
    
    // Get reference voltage
    uint16_t vref_mv = adc_input_get_millivolts(ADC_INPUT_VREF);
    
    PICO_LOGI("ADC", "PTC: %u (%.1fmV), VREF: %umV", 
              ptc_adc, (float)ptc_mv, vref_mv);
    
    // Get full channel data
    adc_input_channel_data_t data;
    if (adc_input_get_channel(ADC_INPUT_AMBIENT_TEMP, &data) == PICO_OK) {
        PICO_LOGI("ADC", "Ambient - Raw: %u, Filtered: %u, mV: %u",
                  data.adc_raw, data.adc_filtered, data.millivolts);
    }
}

Configurationâ€‹

The ADC driver uses configuration values from config.h:

DefinePurpose
CONFIG_PIN_AMBIENT_TEMPGPIO pin for NTC sensor
CONFIG_PIN_HEATER_TEMPGPIO pin for PTC sensor
CONFIG_PIN_VREFGPIO pin for voltage reference
CONFIG_ADC_TIMER_PERIOD_MSSampling interval in milliseconds

Thread Safetyâ€‹

All public functions are protected by a FreeRTOS mutex, making them safe to call from multiple tasks. The internal sampling timer runs independently and updates the channel data atomically.
