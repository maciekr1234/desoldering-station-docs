PID Controller

Proportional-Integral-Derivative controller for temperature regulation.

Overviewâ€‹

The PID driver implements a discrete PID control algorithm used by the HeaterController to maintain precise temperature. It supports both positional and incremental PID modes with anti-windup protection.

Featuresâ€‹

  • Positional and incremental PID modes
  • Configurable integral windup limits
  • Real-time parameter updates
  • Lightweight C implementation

API Referenceâ€‹

Typesâ€‹

typedef enum {
    PID_CAL_TYPE_INCREMENTAL,  // Output is delta change
    PID_CAL_TYPE_POSITIONAL    // Output is absolute value
} pid_calculate_type_t;

typedef struct {
    float kp;            // Proportional gain
    float ki;            // Integral gain
    float kd;            // Derivative gain
    float max_output;    // Output upper limit
    float min_output;    // Output lower limit
    float max_integral;  // Integral upper limit (windup protection)
    float min_integral;  // Integral lower limit (windup protection)
    pid_calculate_type_t cal_type;  // Calculation type
} pid_ctrl_parameter_t;

typedef struct pid_ctrl_block_t* pid_ctrl_block_handle_t;

Example Usageâ€‹

#include "drivers/pid.h"

pid_ctrl_block_handle_t pid;

void setup() {
    pid_ctrl_parameter_t params = {
        .kp = 10.0f, .ki = 0.5f, .kd = 2.0f,
        .max_output = 1.0f, .min_output = 0.0f,
        .max_integral = 50.0f, .min_integral = 0.0f,
        .cal_type = PID_CAL_TYPE_POSITIONAL
    };
    
    pid_new_control_block(&params, &pid);
}

void control_loop() {
    float error = target_temp - current_temp;
    float output;
    
    if (pid_compute(pid, error, &output) == PICO_OK) {
        // Output is 0.0 to 1.0, use for PWM duty
        set_heater_pwm(output);
    }
}

void cleanup() {
    pid_del_control_block(pid);
}
