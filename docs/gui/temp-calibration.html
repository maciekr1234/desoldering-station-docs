Temperature Calibration

Guided wizard for calibrating temperature sensor readings.

Overviewâ€‹

The temperature calibration module provides a step-by-step wizard for calibrating the heater temperature sensor at 6 fixed points (100Â°C, 200Â°C, 300Â°C, 350Â°C, 400Â°C, 450Â°C). Users compare the displayed temperature against an external reference thermometer and enter the actual measured values.

Featuresâ€‹

  • Preview of current calibration values before starting
  • Two calibration methods:
    • Wizard Mode â€” Guided step-by-step with automatic heating and stability detection
    • Manual Entry â€” Direct offset input without heating (for users who already know their offsets)
  • Real-time temperature display during calibration
  • Stability detection (Â±5Â°C for 2 seconds) before allowing input
  • 5-minute timeout per step with automatic cooldown
  • Cancel confirmation dialog
  • Input validation (target Â±50Â°C range)
  • Results summary with Save/Start Over/Exit options

Usageâ€‹

Include the header in your settings screen:

Calibration Methodsâ€‹

Wizard Mode (Recommended)â€‹

The guided wizard heats to each calibration point, waits for stability, and prompts for the actual measured temperature from an external reference thermometer.

Manual Entry Modeâ€‹

For users who already know their calibration offsets (e.g., from a previous calibration or datasheet), the Manual Entry option allows direct input of all 6 offset values without waiting for the heater to reach temperature.

  1. Click "Manual Entry" on the calibration start screen
  2. Enter the offset values for each temperature point (100Â°C through 450Â°C)
  3. Click "Save" to apply the values, or "Back" to cancel
When to Use Manual Entry
  • Re-entering previously known calibration values
  • Testing different offset configurations quickly
  • When external reference thermometer is not available but approximate values are known

Error Handlingâ€‹

Timeoutâ€‹

If temperature doesn't stabilize within 5 minutes, the heater is turned off and the user is returned to the menu after a 2-second cooldown message.

Cancelâ€‹

User can cancel at any time via the Cancel button. A confirmation dialog prevents accidental cancellation. All progress is lost on cancel.

Input Validationâ€‹

Measured temperature input is clamped to target Â±50Â°C (INPUT_RANGE) to prevent obvious entry errors.
