Introduction

Welcome to the Desoldering Station Firmware documentation.

This project is a firmware for an RP2040-based (Raspberry Pi Pico) desoldering station, featuring:

  • FreeRTOS for real-time task management
  • LVGL for a modern graphical user interface
  • PID temperature control for precise heating
  • LittleFS for persistent settings storage

Settings Managementâ€‹

The gSettings global structure holds all runtime configuration, protected by a FreeRTOS mutex. Settings persist to LittleFS as JSON.
