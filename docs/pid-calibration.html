PID Calibration Guide

A practical guide to tuning the PID controller for optimal heater temperature control.

Overviewâ€‹

The desoldering station uses a PID (Proportional-Integral-Derivative) controller to maintain precise heater temperature. Proper tuning ensures:

  • Fast response â€” reaches target temperature quickly
  • Minimal overshoot â€” doesn't exceed target significantly
  • Stable control â€” no oscillations around setpoint
  • Good disturbance rejection â€” recovers quickly after desoldering

Default Valuesâ€‹

The firmware ships with tested default values that work well for most setups:

Start with the defaults. Only tune if you observe issues like slow response, excessive overshoot, or temperature oscillation.

Calibration Methodsâ€‹

Method 1: Manual Tuning (Recommended)â€‹

A practical approach for the desoldering station:

Step 1: Start with P-onlyâ€‹

  1. Set ki = 0, kd = 0
  2. Start with kp = 5.0
  3. Set target temperature to 300Â°C
  4. Observe response:

    • If oscillating â†’ decrease kp
  5. Find the largest kp that gives fast response without sustained oscillation

  2. Start with ki = 0.1

  4. If there's a steady offset â†’ increase ki slightly
  5. If oscillating slowly â†’ decrease ki

Step 3: Add Derivativeâ€‹

  1. Start with kd = 1.0
  2. Observe overshoot when heating to 300Â°C:
    • Significant overshoot â†’ increase kd
    • Very slow response â†’ decrease kd
  3. Fine-tune for minimal overshoot with acceptable response time

Integral Windup Protectionâ€‹

The imax and imin parameters prevent integral windup â€” when the integral term grows excessively large during prolonged errors (like heating from cold).

Recommended Valuesâ€‹

ParameterValueRationale
imax50.0Allows enough integral action to eliminate offset
imin0.0Prevents negative integral (heater can only add heat)
tip

If temperature overshoots significantly after heating from cold, try reducing imax to 30.0 or 40.0.

Troubleshootingâ€‹

Temperature oscillates continuouslyâ€‹

Cause: Kp or Ki too high

Fix:

  1. Reduce kp by 20-30%
  2. If still oscillating, reduce ki by 50%
  3. Increase kd slightly for more damping

Temperature never reaches setpointâ€‹

Cause: Ki too low or Kp too low

Fix:

  1. Verify heater power is sufficient (check PWM output in logs)
  2. Increase ki by 0.1 increments
  3. If very slow, also increase kp

Large overshoot when heatingâ€‹

Cause: Integral windup or Kd too low

Fix:

  1. Reduce imax to 30.0
  2. Increase kd for more damping
  3. Consider reducing ki slightly

Slow response to temperature changesâ€‹

Cause: Gains too conservative

Fix:

  1. Increase kp by 20%
  2. Verify response improves
  3. Add small amount of kd if overshoot appears

Erratic temperature readingsâ€‹

Cause: Sensor noise being amplified

Fix:

  1. Reduce kd â€” derivative term amplifies noise
  2. Check sensor connections
  3. Verify ADC filtering is working (see ADC documentation)
